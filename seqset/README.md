# Multiple domestic measles importations within a two-week period, Washington State, January 2026

## Pathoplexus SeqSet

### Sequence Accessions Files

The full SeqSet is available at https://pathoplexus.org/seqsets/PP_SS_1981.1

`background.txt` includes the accessions for all non-outbreak sequences. These are the sequences that were publicly available at the time of analysis, and passed the inclusion criteria of the phylogenetic workflow.

`focal.txt` includes the list of accessions for sequences from Washington cases during the outbreak investigation.
